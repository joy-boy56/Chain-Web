## üöÄ About the Project

Chain Web is a turn-based strategy game where players place orbs in a grid.  
When a cell reaches its critical mass, it explodes and spreads to adjacent cells ‚Äî potentially triggering cascading reactions.

The objective is simple:
> Eliminate your opponent by taking control of all cells on the board.

This project was developed to:
- Recreate a classic gameplay mechanic for modern browsers
- Improve responsiveness compared to older mobile versions
- Practice DOM manipulation and state management in JavaScript
- Build a complete deployable web project

---

## üß† Gameplay Rules

- Players take turns placing orbs in empty or owned cells.
- Each cell has a maximum capacity depending on its position:
  - Corner cells: 2
  - Edge cells: 3
  - Middle cells: 4
- When the limit is reached:
  - The cell explodes
  - Orbs are distributed to adjacent cells
  - Ownership may change
- Chain reactions can occur.
- The player who eliminates all opponent cells wins.
